RM-Portal-de-Investimento
Desenvolvido por Rafael Melchioretto

e-mail: melchiorettorafael@gmail.com

Especificação O Portal de Investimento é um Sistema Web de Transações Financeiras, otimizanado o trabalho nessas instituições, integrando seus departamentos e atividades. Desde as tarefas administrativas até assistenciais integram esse programa.

Tencologias: Sistema desenvolvido em linguagem PHP Integrado com Banco de Dados MySql Utilizando o Bootstrap, Javascript, HTML e CSS para criação da interface gráfica.

O sistema deve ser descompactado no disco local;

O sistema está vinculado a um banco de dados de teste, para que ele esteja conectado sempre que você for utilizá-lo você deve criar um servidor no banco de dados PostgreSQL e criar um novo banco de dados para armazenar suas informações.
